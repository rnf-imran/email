EmailComposer Plugin Demo

The EmailComposer plugin allows you to initiate an email message draft which can be pre-filled by your app.

After you've checked to make sure the device has an email app and account configured, you can compose a draft - including a list of cc’s, bcc’s and attachments. Even HTML email is supported.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/Wp8 - App package.

Note that the success callback will be invoked when focus has given back to your app (even if an email was not sent). Also note that WP8 doesn't like HTML email.
